# 📝 To-Do Manager (Streamlit)

A simple and interactive To-Do Manager built using Python and Streamlit. It allows users to organize daily tasks efficiently with an easy-to-use web interface.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Search tasks
- View pending and completed tasks
- Store tasks locally using JSON
- Simple and responsive Streamlit UI

## 🛠️ Technologies Used

- Python
- Streamlit
- JSON

## 📂 Project Structure

```
Todo_Manager_Streamlit/
│
├── app.py
└── tasks.json
```

## ▶️ How to Run

1. Install Streamlit

```bash
pip install streamlit
```

2. Run the application

```bash
streamlit run app.py
```
